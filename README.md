## Build

**Clone** - Clone the smart contract to your local machine

```sh
git clone http://github.com/atomizexyz/fenix
```

**Clean** — Clean the build

```sh
forge clean
```

**Build** — Build the smart contract for deployment and testing

```sh
forge build
```

**Test** — Run unit tests

```sh
forge test
```

**Lint** — 

```sh
yarn lint
```
